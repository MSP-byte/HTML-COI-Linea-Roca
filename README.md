# HTML COI Línea Roca

Plano Interactivo para la Gestión Integral de Obras y Servicios de la Línea General Roca.

Sistema desarrollado íntegramente en HTML, CSS y JavaScript puro, sin backend, orientado al seguimiento operativo de Órdenes de Compra, certificaciones, vencimientos, documentación contractual y planificación de obras y servicios.

## Funcionalidades

- Dashboard COI
- Gestión de Obras
- Gestión de Servicios
- Órdenes de Compra (OC)
- Fichas individuales de OC
- Calendario Operativo
- Certificaciones automáticas
- Vencimientos contractuales
- Indicadores tipo semáforo
- Plano interactivo de la Red Línea Roca
- Dashboard por estación
- Buscador multicriterio
- Exportación CSV
- Persistencia mediante localStorage
- Historial de versiones
